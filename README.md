# Veynote

A minimalist note-taking web app with Persian (Solar Hijri) calendar, dark/light themes, and full RTL support.

## Features

- **Notes** — Create, edit, pin, delete notes with auto-save
- **Persian Calendar** — Solar Hijri calendar with event support
- **Search** — Full-text search with instant filtering
- **Media** — Attach images, audio, and video to notes
- **Themes** — Dark & light mode, grayscale palette
- **Fonts** — Vazirmatn (default) and Iranian Sans
- **RTL / LTR** — Persian and English language support
- **Time Format** — 12h / 24h toggle
- **Offline** — Works offline with localStorage persistence
- **Animations** — Smooth page transitions, staggered card entrance, circular editor reveal

## Usage

Open `index.html` in any modern browser. No build step or server required — it's pure vanilla HTML, CSS, and JavaScript.

## Tech Stack

- Vanilla HTML / CSS3 / JavaScript (no frameworks)
- LocalStorage for persistence
- Persian calendar conversion (Solar Hijri)
